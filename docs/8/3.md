# 曲面方程与空间曲线方程的概念
考虑三元方程
$$F (x, y, z) = 0$$

若曲面 $S$ 与这个三元方程有下述关系
1. 曲面 $S$ 上任何一点都满足此方程
2. 不在曲面 $S$ 上的点都不满足此方程
则方程 $(3-1)$ 叫做曲面 $S$ 的方程，曲面 $S$ 叫做方程 $(3-1)$ 的图形
空间曲线可以看作两个曲面 $S _1$ 和 $S_2$ 的交线，即
$$\left\{\begin{matrix} F (x, y, z) = 0 \\ G (x, y, z) = 0 \\ \end{matrix} \right.$$

若曲线 $C$ 与这个三元方程组有下述关系
1. 曲线上的 $C$ 上的任意一点满足方程 $(3-2)$
2. 不在曲线 $C$ 上的点，也不同时在两个平面上
则方程 $(3-1)$ 叫做曲面 $S$ 的方程，曲面 $S$ 叫做方程 $(3-1)$ 的图形

# 平面的点法式方程
- 名词 法向量
平面的法向量垂直于平面中的所有向量

设平面的法向量为 $\mathbf n = (A, B, C)$ ，$\overrightarrow{M _0 M} = (x - x _0, y - y _0, z - z _0)$ 则

$$A (x - x _0) + B (y - y _0) + C (z - z _0) = 0$$

- 名词 平面的点法式方程

# 平面的一般方程
设有三元一次方程
$$ A x + B y + C z = 0 $$

任取满足此方程的一组数 $x _0$ ， $y _0$ ， $z _0$ ，即
$$ A x _0 + B x _0 + C x _0 = 0 $$

两式相减，得
$$ A (x - x _0) + B (y - y _0) + C (z - z _0) = 0 $$

- 名词 平面的一般方程

- 名词 平面的截距方程

$$ \frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1 $$

# 两平面的夹角
- 名词 两平面夹角

平面夹角 $\theta$ 可按以下公式确定

$$ \cos \theta = \frac{ |A _1 A _2 + B _1 B _2 + C _1 C _2| }{ \sqrt{A _1 ^2 + B _1 ^2 + C _1 ^2} \sqrt{A _2 ^2 + B _2 ^2 + C _2 ^2} } $$

两平面互相垂直即 $A _1 A _2 + B _1 B _2 + C _1 C _2 = 0$

不考虑分母为零的情况，两平面互相平行或重合即
 $$\frac{A _1}{A _2} = \frac{B _1}{B _2} = \frac{C _1}{C _2}$$
