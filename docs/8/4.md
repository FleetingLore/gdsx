# 两直线的夹角
- 名词  两直线的夹角
两直线的夹角可由以下公式确定
$$ \cos \varphi = \frac{ |m _1 m _2 + n _1 n _2 + p _1 p _2| }{ \sqrt{m _1 ^2 + n _1 ^2 + p _1 ^2} \sqrt{m _2 ^2 + n _2 ^2 + p _2 ^2} } $$

# 直线与平面的夹角
`名词` 直线与平面的夹角
直线和平面的夹角可由以下公式确定
$$ \sin \varphi = \frac{ |A m + B n + C p| }{ \sqrt{A ^2 + B ^2 + C ^2} \sqrt{m ^2 + n ^2 + p ^2} }$$
---
不考虑分母为零，直线与平面垂直相当于
$$\frac{A}{m} = \frac{B}{n} = \frac{C}{p}$$
---
直线与平面平行或直线在平面上即
$$A m + B n + C p = 0$$


# 杂例
设直线由以下方程组确定
$$\left\{ \begin{matrix} A _1 x + B _1 y + C _1 z + D _1 = 0 \tag{4-11} \\ A _2 x + B _2 y + C _2 z + D _2 = 0 \\ \end{matrix} \right. $$
可知系数不成比例
则对于任意的 $\lambda$ ，给出经过这个直线的一个由 $\lambda$ 确定的平面
$$A _1 x + B _1 y + C _1 z + D _1 + \lambda (A _2 x + B _2 y + C _2 z + D _2) = 0$$

- 名词 平面束

- 名词 平面束方程
