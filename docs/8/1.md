# 向量的线性运算
### 向量的加减法

- 加法
$$\mathbf c = \mathbf a + \mathbf b$$

- 加法交换律
$$\mathbf a + \mathbf b = \mathbf b + \mathbf a$$

- 加法结合律
$$(\mathbf a + \mathbf b) + \mathbf c = \mathbf a + (\mathbf b + \mathbf c)$$

- 取反
$$-\mathbf a$$

- 减法
$$\mathbf a - \mathbf b = \mathbf a + (- \mathbf b)$$

- 三角不等式
$$\mid \mathbf a + \mathbf b \mid \le \mid \mathbf a \mid + \mid \mathbf b \mid$$

### 向量与数的乘法

- 数乘
规定 $\lambda \mathbf a$ 是一个向量
$$\mid \lambda \mathbf a \mid = \mid \lambda \mid \mid \mathbf a \mid$$
当 $\lambda > 0$ 时与 $\mathbf a$ 同向
当 $\lambda < 0$ 时与 $\mathbf a$ 反向.
当 $\lambda = 0$ 时为 $\mathbf 0$.

- 数乘结合律
$$\lambda (\mu \mathbf a) = \mu (\lambda \mathbf a) = (\lambda \mu) \mathbf a$$

- 数乘分配律
$$(\lambda + \mu)\mathbf a = \lambda \mathbf a + \mu \mathbf a \tag{1-1}$$
$$\lambda(a + b) = \lambda a + \lambda b \tag{1-2}$$

- 名词 线性运算
向量的相加和数乘运算的统称

- 标记 $\mathbf e _a$
与 $\mathbf a$ 方向相同的单位向量

- 定理
设向量 $a \ne 0$ ,则向量 $b$ 平行于 $a$ 的充分必要条件是:存在唯一的实数 $\lambda$ ,使 $\mathbf b = \lambda \mathbf a$

这个定理的证明,分为充分性和必要性.
充分性是显然的,必要性的证明包括 $\lambda$ 的唯一性证明.

# 利用直角坐标系作向量的线性运算
- 定比分点
已知点 $A(x_1, y_1, z_1)$ 和 $B(x_2, y_2, z_2)$ 以及实数 $\lambda \ne -1$ 在直线 $AB$ 上求点 $M$ ,使
$$\overrightarrow{AM} = \lambda \overrightarrow{MB}$$
有
$$M\left(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}, \frac{z_1 + z_2}{2}\right)$$

# 向量的模、方向角、投影
### 向量的模与两点间的距离公式

- 在空间直角坐标中  向量的模
$$\mid \mathbf r \mid = \mid OM \mid = \sqrt{\mid OP \mid ^2 + \mid OQ \mid ^2 + \mid OR \mid ^2} = \sqrt{x^2 + y^2 + z^2}$$

- 在空间直角坐标系中  两点间距离
$$\mid AB \mid = \mid \overrightarrow{AB} \mid = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2 + (z_1 - z_2)^2}$$

### 方向角与方向余弦

- 名词
方向角
方向余弦

- 空间直角坐标系中  方向角和方向余弦的关系
$$\cos \alpha = \frac{x}{r}, \cos \beta = \frac{y}{r}, \cos \gamma = \frac{z}{r}$$

- 空间直角坐标系中  方向余弦的独立性
$$\cos ^2 \alpha + \cos ^2 \beta + \cos ^2 \gamma = 1$$

### 向量在轴上的投影

- 表述
点 $M$ 在 $u$ 轴的投影
向量 $\mathbf r$ 在 $u$ 轴上的投影

- 标记
$\text{Prj} _u \mathbf r$
$(\mathbf r) _u$

- 向量的投影具有与坐标相同的性质
$$\text{Prj} _u \mathbf a = \mid \mathbf a \mid \cos \varphi$$
其中 $\varphi$ 为向量 $\mathbf a$ 与 $u$ 轴的夹角
---
$$\text{Prj} _u (\mathbf a + \mathbf b) = \text{Prj} _u \mathbf a + \text{Prj} _u \mathbf b$$
---
$$\text{Prj} _u (\lambda \mathbf a) = \lambda \text{Prj} _u \mathbf a$$
