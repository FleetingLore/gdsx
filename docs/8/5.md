### 椭圆锥面

- 名词 截痕法

### 椭球面
$$\frac{x ^2}{a ^2} + \frac{y ^2}{b ^2} + \frac{z ^2}{c ^2} = 1$$

- 名词 旋转椭球面

### 单叶双曲面
$$\frac{x ^2}{a ^2} + \frac{y ^2}{b ^2} - \frac{z ^2}{c ^2} = 1$$

### 双叶双曲面
$$\frac{x ^2}{a ^2} - \frac{y ^2}{b ^2} - \frac{z ^2}{c ^2} = 1$$

### 椭圆抛物面

- 名词 旋转抛物面
