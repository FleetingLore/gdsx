# 11-4  对面积的曲面积分

## 1. 对面积的曲面积分的概念和性质

> `定义`  对面积的曲面积分
>
> $$\iint _\Sigma f (x, y,z) \text{d} S = \lim _{\lambda \to 0} \sum _{i = 1} ^n f (\xi _i, \eta _i, \zeta _i) \Delta S _i$$

## 2. 对面积的曲面积分的计算法

$$\iint _\Sigma f (x, y,z) \text{d} S
= \iint _{D _{x y}} f [x, y, z (x, y)] \sqrt{1 + z _x ^2 (x, y) + z _y ^2 (x, y)} \text{d} x \text{d} y$$
