# 映射

# 映射概念

#[定义] 映射 {
  X, Y: 非空集合

  对 X 中的每个元素 x, 按法则 f,在 Y 中有唯一确定的 y 与之对应 {
    f: X 到 Y 的映射
    f: X to Y

    #[名词]
    y {
      元素 x 在映射 f 下的像
    } 像

    #[名词]
    x {
      元素 y 在映射 f 下的原像
    } 原像

    y = f(x)

    #[名词]
    X {
      映射 f 的定义域
    } 定义域 [
      D[f]
    ]

    #[名词]
    Y {
      映射 f 的值域
    } 值域 [
      R[f]
      f(X)
      {f(x) mid x in X}
    ]
  }
}

#[名词]
满射 | X 到 Y 上的映射 {
  f: X to Y
  Y 中任一元素 y 都是 X 中某元素的像
}

#[名词]
单射 {
  f: X to Y
  x[1], x[2]: x[1] ne x[2]
  f(x[1]) ne f(x[2])
}

#[名词]
双射 | 一一映射 {
  f: X to Y
  满射
  单射
}

# 逆映射与复合映射

#[名词]
逆映射 {
  f: Y to X
  f: 单射
  f(-1): R[f] to X
}

D[f(-1)] = R[f]
R[f(-1)] = X

只有单射才存在逆映射

#[名词]
复合映射 {
  g: X to Y[1]
  f: Y[2] to Z
  Y[1] subset Y[2] | R[g] subset D[f] {
    f circ g: X to Z
    (f circ g)(x) = f(g(x))
    x in X
  }
}

f circ g 有意义不代表 g circ f 有意义
即使 f circ g 和 g circ f 都有意义,两者也未必相同

# 函数的概念

#[定义]
函数 {
  D: 数集
  D subset mathbb R
  f: D to mathbb R
  f 为定义在 D 上的函数

  #[名词]
  y = f(x) {
    #[名词] {
      x in D
    } 因变量
  } 自变量

  #[名词] {
    D
    D[f]
  } 定义域
}

#[名词] {
  对每个 x in D,按对应法则 f,总有唯一确定的值 y 与之对应
  这个值称为函数 f 在 x 处的函数值
} 函数值

#[名词] {
  因变量 y 与自变量 x 之间的这种依赖关系
} 函数关系

#[名词] {
  函数值 f(x) 的全体所构成的集合
  R[f]
  f(D)
  {y mid y = f(x), x in D}
} 值域

#[名词] {
  定义域 {
    使得解析式有意义的一切实数组成的集合
  }
} 自然定义域

表示函数的主要方法 {
  表格法
  解析法
  图形法
}

#[名词] {
  坐标平面上的点集 {P(x, y) mid y = f(x), x in D}
  函数 f 在 D 上的图形
} 图形
