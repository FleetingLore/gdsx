# 重积分的应用

## 1. 曲面的面积

$$A = \iint _D \sqrt{1 + \left( \frac{\partial z}{\partial x} \right) ^2 + \left( \frac{\partial z}{\partial y} \right) ^2} \text{d} x \text{d} y$$

## 2. 质心

$$\overline{x} = \frac{M _y}{M} = \frac{\displaystyle \iint _D x \mu (x, y) \text{d} \sigma}{\displaystyle \iint _D \mu (x, y) \text{d} \sigma}$$

## 3. 转动惯量

$$I _x = \sum _{i = 1} ^n y _i ^2 \mu _i$$
