# 线性微分方程的解的结构

讨论二阶齐次线性方程

$$y''+ P(x)y' + Q(x)y = 0 \tag{6-6}$$

`定理` 如果函数 $y_1(x)$ 与 $y_2(x)$ 是此方程的两个解, 那么

$$y = C_1y_1(x) + C_2y_2(x)$$

也是此方程的解, 其中 $C_1$, $C_2$ 为任意常数.

这未必表达了通解.

---

`名词` 设 $y_1(x)$, $y_2(x)$, $\cdots$, $y_n(x)$ 为定义在区间 $I$ 上的 $n$ 个函数, 如果存在 $n$ 个不全为零的常数 $k_1$, $k_2$, $\cdots$, 使得当 $x\in I$ 时有恒等式

$$k_1y_1 + k_2y_2 + \cdots + k_ny_n = 0$$

成立, 那么称这 $n$ 个函数在区间 $I$ 上**线性相关**; 否则称**线性无关**.

---

`定理` 如果 $y_1(x)$ 与 $y_2(x)$ 是方程(6-6)的两个线性无关的解,那么

$$y=C_1y_1(x) + C_2y_2(x)\ (C_1,C_2为任意常数)$$

就是方程(6-6)的通解.

---

`定理` 如果 $y_1(x)$, $y_2(x)$, $\cdots$, $y_n(x)$ 是 $n$ 阶齐次线性方程

$$y^{(n)} + a_1(x)y^{(n-1)} + \cdots + C_ny_n(x) = 0$$

的 $n$ 个线性无关的解,那么,此方程的通解为

$$y=C_1y_1(x) + C_2y_2(x) + \cdots + C_ny_n(x)\ ,$$

其中 $y_1(x)$, $y_2(x)$, $\cdots$, $y_n(x)$ 为任意常数

---

`定理` 设 $y^*$ 是二阶非齐次线性方程

$$y'' + P(x)y' + Q(x)y = f(x) \tag{6-5}$$

的一个特解.$Y(x)$是与(6-5)对应的齐次方程(6-6)的通解,则

$$y=Y(x) + y^*(x) \tag{6-8}$$

是非齐次线性微分方程(6-5)的通解.

---

`定理` 设非齐次线性方程(6-5)的右端 $f(x)$ 是两个函数之和,即

$$y''(x) + P(x)y' + Q(x)y = f_1(x) + f_2(x)$$

而 $y_1^*(x)$ 与 $y_2^*(x)$ 分别是方程

$$y''+P(x)y'+Q(x)y=f_1(x)$$

与

$$y''(x)+P(x)y'+Q(x)y=f_2(x)$$

的特解,则 $y_1^*(x) + y_2^*(x)$ 就是原方程的特解.

$2$ 改 $n$ 同理.
