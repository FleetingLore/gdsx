# 微分方程的基本概念

`名词` 一般地, 凡表示未知函数、未知函数的导数与自变量之间的关系的方程,叫做**微分方程**.

微分方程中所出现的非直属的最高阶导数的阶数, 叫做**微分方程的阶**.

$n$ 阶微分方程的形式是

$$F(x,\ y,\ y',\ \cdots ,\ y^{(n)})$$

---

`名词` 找出这样的函数, 把这函数带入微分方程能使该方程成为恒等式, 这个函数就叫做该**微分方程的解**.

如果在区间 $I$ 上,

$$f\left[x,\ \varphi(x),\ \varphi'(x),\ \cdots,\ \varphi^{(n)}\right] \equiv 0$$

那么函数 $y=\varphi(x)$ 叫做**微分方程在区间 $I$ 上的解**.

如果微分方程的解中含有任意常数, 且任意常数的个数与微分方程的阶数相同, 这样的解叫做**微分方程的通解**.

---

`名词` 借助**初值条件**, 确定了通解中的任意常数以后, 就得到微分方程的特解.

求微分方程 $y'=f(x,\ y)$ 满足初值条件 $y\mid_{x=x_0}=y_0%$ 的特解这样一个问题, 叫做一阶微分方程的**初值问题**.
